# Настройка SEO плагинов для публикации статей

## Поддерживаемые плагины

- **Rank Math SEO**
- **Yoast SEO**
- **Оба плагина одновременно**

## Автоматическое определение плагина

Программа автоматически определяет установленный SEO плагин на основе настройки `seo_plugin` в `config_multisite.py`:

```python
"seo_plugin": "rankmath"  # только Rank Math
"seo_plugin": "yoast"     # только Yoast SEO
"seo_plugin": "both"      # оба плагина
```

## Установка кода в WordPress

### Для Rank Math SEO

1. Откройте файл `functions.php` активной темы WordPress
2. Добавьте код из файла `functions_rankmath_snippet.php`
3. Сохраните файл

### Для Yoast SEO

1. Откройте файл `functions.php` активной темы WordPress
2. Добавьте код из файла `functions_yoast_snippet.php`
3. Сохраните файл

### Для обоих плагинов

1. Откройте файл `functions.php` активной темы WordPress
2. Добавьте код из файла `functions_both_seo_snippet.php`
3. Сохраните файл

## Что делает код

Код выполняет две функции:

1. **Регистрирует мета-поля в REST API** - позволяет передавать SEO данные через WordPress REST API
2. **Синхронизирует данные** - сохраняет переданные через API данные в реальные мета-поля плагина

## Проверка работы

После добавления кода в `functions.php`:

1. Создайте статью через интерфейс
2. Проверьте в WordPress админке:
   - Для Rank Math: раздел "Rank Math SEO" в редакторе поста
   - Для Yoast SEO: блок "Yoast SEO" в редакторе поста
3. Должны быть заполнены поля:
   - **SEO Title** (meta_title)
   - **Meta Description** (meta_description)

## Важно

- Код должен быть добавлен в `functions.php` **активной темы**
- После добавления кода перезагрузите сайт (или очистите кеш, если используется)
- Убедитесь, что у пользователя WordPress есть права на редактирование постов
