# Интерфейс публикации статей

Интерфейс для публикации статей создан на основе компонентов из `news` и `translations`.

## Структура

```
src/articles/
├── ArticlesPage.jsx          # Главная страница со списком статей
├── api/
│   └── articlesService.js    # API сервис для работы со статьями
└── components/
    ├── ArticlesTable.jsx     # Таблица со статьями
    └── ArticlePublishModal.jsx  # Модальное окно для публикации
```

## Установка и запуск

### 1. Установите Node.js

Если Node.js не установлен, скачайте и установите с [nodejs.org](https://nodejs.org/)

### 2. Установите зависимости

```bash
npm install
```

### 3. Запустите dev сервер

```bash
npm run dev
```

Приложение откроется в браузере по адресу `http://localhost:5173`

## Использование

1. Перейдите на страницу `/articles` в приложении
2. Просматривайте список статей в таблице
3. Нажмите "Опубликовать" для публикации статьи
4. Выберите окружение (test/production) и подтвердите публикацию

## Особенности

- ✅ Таблица со статьями с пагинацией
- ✅ Модальное окно для публикации
- ✅ Поддержка темной темы
- ✅ Адаптивный дизайн
- ✅ Интеграция с существующими компонентами (Loader, Pagination)

## API

Сервис `articlesService.js` использует переменные окружения:
- `VITE_ARTICLES_API_BASE` - базовый URL API для статей
- `VITE_ENDPOINT` - общий endpoint (используется как fallback)

Если API недоступен, сервис возвращает пустые данные для демонстрации интерфейса.
